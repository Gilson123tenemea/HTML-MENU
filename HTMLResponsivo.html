Diseño web responsivo HTML


¡El diseño web responsivo consiste en crear páginas web que se vean bien en todos los dispositivos!

Un diseño web responsivo se ajustará automáticamente a diferentes tamaños de pantalla y ventanas gráficas.




¿Qué es el diseño web responsivo?

El diseño web responsivo implica utilizar HTML y CSS para redimensionar, ocultar, reducir o ampliar automáticamente un sitio web para que se vea bien en todos los dispositivos (computadoras de escritorio, tabletas y teléfonos):
